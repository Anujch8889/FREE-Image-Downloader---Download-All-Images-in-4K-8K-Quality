## üîß Troubleshooting

### Problem: Extension doesn't appear after loading
**Solution**: Refresh the extensions page (`chrome://extensions/`) and check if it's listed

### Problem: "Scan For Images" button doesn't work
**Solution**: 
1. Make sure you're on an actual webpage (not chrome:// pages)
2. Refresh the webpage and try again
3. Check browser console for errors (F12)

### Problem: No images found
**Solution**: 
- Some pages may not have traditional images
- Try a different website
- Ensure the page has fully loaded

### Problem: Downloads don't start
**Solution**:
1. Check Chrome download settings
2. Make sure downloads aren't blocked
3. Check if download permission is granted
